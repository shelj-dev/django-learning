Getting started

Commands to create a Django project using virtual environment:

Terminal
# create venv (example)
python -m venv venv

# Virtual environment run command
# source venv/bin/activate             # linux / mac
venv\Scripts\activate                  # windows

# install django
pip install django

# create project
django-admin startproject mysite
cd mysite

# run development server
python manage.py runserver

Create app

Where the project files and logic lives.

Terminal
python manage.py startapp core

Project layout:

mysite/
├─ manage.py
├─ mysite/                # project settings
│  ├─ __init__.py
│  ├─ asgi.py
│  ├─ settings.py
│  ├─ urls.py
│  └─ wsgi.py
└─ core/          # app
   ├─ __init__.py
   ├─ admin.py
   ├─ apps.py
   ├─ models.py
   ├─ tests.py
   └─ views.py
