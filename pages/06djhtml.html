Django HTML (Static & Dynamic Templates)

Learn how HTML works inside Django using templates and Python logic

Django uses a powerful template engine that allows you to mix HTML with Python-like syntax safely. This lesson explains static HTML, dynamic content, template tags, and filters.

Step 1: Static HTML in Django

Static HTML files are placed inside the templates/ folder. 

myapp/templates/home.html

<h1>Welcome to Django</h1>
<p>This is a static HTML page.</p>

Step 2: Rendering HTML from Views

myapp/views.py

from django.shortcuts import render

def home(request):
    return render(request, "home.html")

Step 3: Dynamic HTML (Passing Data)

You can send Python data from views to HTML templates using context. 

myapp/views.py

def dashboard(request):
    context = {
        "username": "Shelj",
        "age": 22
    }
    return render(request, "dashboard.html", context)

myapp/templates/dashboard.html

<h2>Dashboard</h2>
<p>Name: {{ username }}</p>
<p>Age: {{ age }}</p>

Step 4: Python Logic inside HTML

Django provides template tags for logic like loops and conditions. 

Example

<ul>
    {% for item in items %}
        <li>{{ item }}</li>
    {% endfor %}
</ul>

If Condition

{% if is_logged_in %}
    <p>Welcome back!</p>
{% else %}
    <p>Please log in</p>
{% endif %}

Step 5: Template Inheritance

Reuse HTML using a base template.

myapp/templates/base.html

<html>
<body>
    <header>My Site</header>

    {% block content %}
    {% endblock %}

    <footer>Footer</footer>
</body>
</html>

child.html

{% extends "base.html" %}

{% block content %}
<h1>Child Page</h1>
{% endblock %}

Step 6: Static Files (CSS & JS)

HTML

{% load static %}
<link rel="stylesheet" href="{% static 'css/style.css' %}">

Important Notes

  • Django templates do NOT allow raw Python code
  • Use {{ }} for variables
  • Use {% %} for logic
  • Always escape HTML automatically (secure)
