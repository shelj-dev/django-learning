Django HTML (Static & Dynamic Templates)

Learn how HTML works inside Django using templates and Python logic

Django uses a powerful template engine that allows you to mix HTML with Python-like syntax safely. This lesson explains static HTML, dynamic content, template tags, and filters.

Step 1: Static Files (CSS & JS)

myapp/home.html

{% load static %}
<link rel="stylesheet" href="{% static 'css/style.css' %}">

myapp/home.html

.....
<img src="{% static 'sun.jpg' %}" alt="" width="550px">

Important Notes

  • Django templates do NOT allow raw Python code
  • Use {{ }} for variables
  • Use {% %} for logic
  • Always escape HTML automatically (secure)
