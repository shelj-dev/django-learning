URLs

Map URLs to views and render templates

urls.py controls which view runs for each URL in website.

myproject/urls.py

# Connect app url with the project.

from django.contrib import admin
from django.urls import path, include

urlpatterns = [
    path('admin/', admin.site.urls),
    path('myapp/', include("myapp.urls")),
]

myapp/urls.py

from django.urls import path
from myapp import views

urlpatterns = [
    path('home/', views.home),
]

Views

A view is simply a Python function or class that receives a request and returns a response. 
myapp/views.py

from django.shortcuts import render


def home(request):
    return render(request, "home.html")
