Django Admin & Superuser

Model registration, superuser creation, admin login, database check

Django provides a built-in Admin Panel to manage data. Only superusers or staff users can access it.

Register Models in Admin

Register models in admin.py to display them in the admin panel. 

myapp/admin.py
from django.contrib import admin
from myapp.models import Student, Course

admin.site.register(Student)
admin.site.register(Course)

Create Superuser

Create a superuser using the terminal. 

Terminal
python manage.py createsuperuser

Example

Admin Login

Open the admin login page: 


http://127.0.0.1:8000/admin/

Login using the superuser username and password.

Admin Dashboard

After login, the admin dashboard appears. You can manage all registered models.

Add, edit, or delete data for Student and Course.

Database Verification

Data added through the admin panel is stored in the database.

This confirms that admin data is saved successfully.
