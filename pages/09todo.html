ToDo app

Create a new project

Terminal
python -m venv .venv
.venv/scripts/activate
pip install django

# adding . at the end create the project in the current folder instead of creating a new folder
django-admin startproject todo . 
django-admin startapp app

│  ├─ settings.py
│  └─ wsgi.py
